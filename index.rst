================================================
Welcome to the DCAT-AP Switzerland documentation
================================================

This page is a technical documentation of the DCAT-AP Switzerland metadata standard to help publishers get their data on the Swiss open data portal.

For publishers, OPENDATA.SWISS offers three ways to add their datasets to the metadata catalogue:

#. :doc:`Use the provided form <form>` to enter all the data (*recommended for ~1-20 datasets*)
#. :doc:`Upload data as RDF/XML <upload>` conforming the DCAT-AP standard (*recommended for ~20-100 datasets*)
#. :doc:`Provide a RDF/XML DCAT catalog endpoint <harvester>` for harvesting (*recommended for >=100 datasets*)

All of the above options make use of the :doc:`DCAT-AP format <dcat-ap-format>`.
